# Summary
Create new agent skill definitions with structured procedure and verification sections.

# Procedure
- Gather the task objective and required tool dependencies.
- Draft the SKILL.md with summary, procedure steps, and verification checks.
- Validate the skill through lint and a dry-run ingest cycle.

# Verification
- Confirm the skill ingests without lint errors and renders a complete snippet.
