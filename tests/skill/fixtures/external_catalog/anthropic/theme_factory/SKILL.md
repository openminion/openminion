# Summary
Generate themed UI configuration files and design token sets for applications.

# Procedure
- Read the base theme definition and identify the customization targets.
- Generate the derived theme file with overridden color, spacing, and typography values.
- Write the output to the project theme directory.

# Verification
- Confirm the generated theme file is valid and the application loads without errors.
