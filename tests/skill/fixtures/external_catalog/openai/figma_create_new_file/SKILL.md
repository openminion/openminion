# Summary
Create a new Figma file from a template with project-specific page structure.

# Procedure
- Select the appropriate Figma template for the project type.
- Create the initial page structure with cover, components, and flows pages.
- Apply the team naming convention and organize layers.

# Verification
- Confirm the new file follows the team template and naming conventions.
