# Summary
General Figma design file inspection and annotation.

# Procedure
- Open the Figma file URL in the browser and navigate to the target frame.
- Inspect layer names, auto-layout constraints, and color tokens.
- Leave structured annotations on the relevant components.

# Verification
- Confirm annotations are attached to the correct layers.
