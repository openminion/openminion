# Summary
Create and maintain design system rules in Figma including tokens and component guidelines.

# Procedure
- Audit the existing Figma library for inconsistent tokens.
- Define color, spacing, and typography scales as design system variables.
- Document component usage rules alongside the Figma source, referencing CLAUDE.md, AGENTS.md, and tailwind.config.js for alignment.
- Export the token set as a Button.tsx example and index.tsx scaffold.

# Verification
- Confirm all design tokens resolve and the component guidelines are complete.
